🍲 Recipe Management Application (WPF)
📖 Overview

The Recipe Management Application is a desktop application built using C#, .NET, and WPF (Windows Presentation Foundation).

This project was developed to demonstrate the application of Object-Oriented Programming (OOP) principles, advanced C# features, and modern WPF UI design.

The system allows users to create, view, scale, and manage recipes through an intuitive graphical interface.

🎯 Objectives

Apply core OOP principles:

Encapsulation

Inheritance

Polymorphism

Abstraction

Build a fully functional WPF GUI application

Implement data handling and logic separation using MVVM

Translate a traditional command-line application into a modern GUI
